# 四柱八字命理学 SCORM 2004 课程包

## 📚 课程简介

本课程是基于《四柱八字.txt》教材开发的完整 SCORM 2004 格式在线课程，系统讲解中国传统四柱八字命理学知识体系。

## 🎯 课程特色

- **6个核心章节**：从基础到实战，循序渐进
- **30+详细例题**：理论联系实际，深入浅出
- **60道测验题目**：巩固知识，检验学习效果
- **传统文化设计**：深红金色配色，彰显中华传统美学
- **完整SCORM追踪**：支持主流LMS系统

## 🚀 使用方法

### 方法一：LMS系统导入

1. **打包课程**
   ```bash
   cd scorm/
   zip -r ../四柱八字课程.zip *
   ```

2. **导入LMS**
   - 登录您的LMS系统（Moodle/Blackboard/Canvas等）
   - 选择"导入SCORM包"
   - 上传 `四柱八字课程.zip`
   - 发布课程

3. **学员学习**
   - 学员登录LMS即可开始学习
   - 系统自动追踪学习进度
   - 完成测验后自动记录成绩

### 方法二：本地预览

```bash
cd scorm/
python3 -m http.server 8000
```

然后在浏览器中访问：`http://localhost:8000/index.html`

**注意**：本地预览时SCORM API无法连接LMS，但不影响浏览课程内容和完成测验。

## 📊 测验说明

### 测验规则
- 每章10道题目（8道单选 + 2道判断）
- 满分100分，及格线60分
- 即时反馈（正确显示绿色，错误显示红色）
- 可重新作答

### 评分标准
- 每题10分
- 答对得10分，答错得0分
- 总分≥60分为通过

## 🔧 技术规范

### SCORM标准
- 遵循SCORM 2004第三版
- 支持cmi数据模型
- 实现完整的API_1484_11接口

### 兼容性
- **浏览器**：Chrome 60+, Firefox 55+, Edge 79+, Safari 11+
- **LMS系统**：Moodle 3.5+, Blackboard Learn, Canvas, TalentLMS
- **编码**：UTF-8（完美支持中文）
- **依赖**：无外部依赖，纯原生JavaScript

### SCORM追踪项
- `cmi.completion_status`: 完成状态
- `cmi.success_status`: 成功状态（通过/失败）
- `cmi.score.raw`: 原始分数
- `cmi.score.scaled`: 标准化分数
- `cmi.session_time`: 会话时长
- `cmi.location`: 当前位置

## 🐛 故障排除

### 问题1：LMS导入失败
- **原因**：manifest文件损坏或格式错误
- **解决**：检查 `imsmanifest.xml` 是否完整

### 问题2：中文显示乱码
- **原因**：编码不是UTF-8
- **解决**：确保所有文件以UTF-8保存

### 问题3：测验无法提交
- **原因**：JavaScript被禁用或浏览器不兼容
- **解决**：启用JavaScript，使用推荐浏览器

### 问题4：SCORM数据未保存
- **原因**：LMS配置问题
- **解决**：联系LMS管理员检查SCORM设置
