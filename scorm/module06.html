第六章：合盘简析

Comprehensive Chart Analysis

学习目标

  • 掌握干支冲合会刑穿在人事应用中的涵义
  • 熟练运用地支冲合会刑穿的掌诀方法
  • 理解四柱八字综合分析的基本原理
  • 学会年月日时干支的相互关系判断
  • 能够进行完整八字的合盘综合分析

一、干支冲合会刑穿用于人事

在进行四柱八字的综合分析之前，必须深入理解干支冲合会刑穿的涵义。四柱八字的结构中，任何一组干支的冲合会刑穿均有可能产生「吉」的后果与「凶」的后果，有时亦往往吉、凶两种征象同时显现。

基本判断原则

论吉：八字所喜用的干支来冲去、合去、刑去或穿害八字所仇忌的干支

论凶：八字所仇忌的干支来冲去、合去、刑去或穿害八字所喜用的干支

（一）合的涵义

包括半合、六合、三合、会方等形式：

吉的方面

  • 合作、合伙、和解
  • 连系、沟通、交友
  • 恋爱、结婚、思念
  • 默契、通信、连络
  • 探访、团聚、会合
  • 结社、集会、组合
  • 改善、由逆转顺
  • 交到好朋友、遇上好搭档

凶的方面

  • 牵绊、制肘、纠缠
  • 束缚、捆绑、监禁
  • 困乏、郁闷、消沉
  • 灰心、拖累、压力
  • 重担、矛盾、分心
  • 犹豫、妥协、投降
  • 由顺转逆
  • 交到坏朋友、遇上坏搭档

（二）冲的涵义

吉的方面

  • 新的转机、脱离束缚
  • 解除牵绊、下决心改善积弊
  • 戒除不良恶习
  • 好的转折点、新创意、新策略
  • 果断的裁定、外来之助力
  • 制服敌对者、克服难题
  • 迁移、升职、出差、出国
  • 创业、病愈、收获、佳音
  • 除旧布新、整修、破土兴工

凶的方面

  • 坏的转折点、猜忌误会
  • 心神不宁、人事不安
  • 动荡不稳、跌打损伤
  • 拆散、分离、打击
  • 降职、调差、逃避、逃亡
  • 异变、是非、冲突、仇怨
  • 碰撞、磨擦、病灾
  • 付出、噩耗、断裂、绝交
  • 既有之物遭外力破坏或掠夺

（三）刑和害的涵义

刑和害的涵义跟「冲」不太一样：

  • 冲：属于直接的、突发的、明显的、相对的，并于较短期间内见真章的吉凶事象
  • 刑与害：多属较为长期性的潜伏因素所蕴酿造成之吉凶事象，其发生有远因可溯

注意：刑与害往往牵涉到命造本人或他人的意识心态、生活习惯及法律规制，可能产生「我刑他害他」或「他刑我害我」的现象。

二、地支冲合会刑穿的掌诀

（一）地支十二宫掌诀

地支掌诀又称「十二宫掌诀」，因其用途甚广，故务须记熟。一般均以左手较为方便。

掌诀类型 组合形式 记忆方法
六冲掌诀 子午、卯酉、巳亥、辰戌、丑未、寅申 用斜线记忆（180度交叉对冲）
六合掌诀 子丑、寅亥、卯戌、辰酉、巳申、午未 用横线记忆（横向直线联结之合）
三合掌诀 寅午戌、亥卯未、巳酉丑、申子辰 用三角形记忆（等腰三角形联结）
会方掌诀 寅卯辰、巳午未、申酉戌、亥子丑 用四方单面串联记忆（气偏一方）
六害掌诀 子未、丑午、寅巳、卯辰、申亥、酉戌 用纵线记忆（上下相对之穿害）
相刑掌诀 寅申巳、辰戌丑未、子卯、午、酉、亥自刑 四角形帮助记忆，四正方位自刑

（二）地支三刑与相破

地支三刑

  • 四长生三刑：亥寅巳、寅巳申、巳申亥、申亥寅（由寅申巳亥四长生之地支而来）
  • 四墓库三刑：未戌丑、戌丑辰、丑辰未、辰未戌（由辰戌丑未四墓库之地支而来）
  • 刑合：亥寅与巳申这两组相刑亦牵涉到六合，既刑又合

特点：三刑的影响力较相刑为大，能使吉者增吉，凶者倍凶

地支相破

所谓的「破」，其原意通「迫」字，即逼迫、迫害、压迫之意：

  • 卯破午：卯中乙木逼迫午中临官之己土
  • 午破酉：午中丁火逼迫酉中辛金
  • 酉破亥：酉中辛金逼迫亥中甲木
  • 子破巳：子中癸水逼迫巳中丙火

三、综合分析方法

（一）四柱之间的相互关系

四柱八字由年、月、日、时四柱组成，每柱天干地支各一字，共计八个字。在进行综合分析时，需要全面考察四柱之间的相互关系：

分析层面 分析要点 注意事项
年柱与月柱 祖业根基与父母宫位的关系，早年运势基础 观察年月天干是否相生相合，地支是否冲刑穿害
月柱与日柱 月令旺衰与日主强弱的关系，青年运势发展 月令对日主的生扶克泄，决定格局高低
日柱与时柱 日主与子女宫位的关系，晚年运势归宿 时柱为归宿之地，关系晚年福祉
年柱与时柱 始终首尾的呼应关系，一生运势起伏 年时天地相冲，主生活动荡多变

（二）天干地支的综合判断

1. 天干分析要点

  • 天干主外、主显、主动，代表外在表现和社会关系
  • 观察天干五合：甲己合化土、乙庚合化金、丙辛合化水、丁壬合化木、戊癸合化火
  • 天干相克：甲克戊己、乙克己戊、丙克庚辛、丁克辛庚、戊克壬癸等
  • 天干透出：地支藏干透出天干，力量增强

2. 地支分析要点

  • 地支主内、主隐、主静，代表内在本质和根基深浅
  • 地支藏人元：每个地支内藏天干，影响五行力量
  • 地支会局：三合局、方局的形成，改变五行格局
  • 地支暗冲暗合：不直接相邻但通过大运流年产生作用

3. 干支综合分析

  • 天干地支同气：如甲寅、乙卯，力量最强
  • 天干地支相生：如甲子、丙寅，相得益彰
  • 天干地支相克：如甲申、丙子，内外矛盾
  • 干支通根：天干在地支有根气，力量稳固

四、完整八字的合盘分析实例

【实例一】子丑相合的综合分析

命例：八字中见到子、丑两字同时出现

基本分析：子与丑极有可能化成「土」

综合判断：

  • 情形一（合化成功）：如果八字全局土行力量适中，又无强水或旺木干扰，则子丑可以合化成土，增强土的力量
  • 情形二（合而不化）：如果八字全局水势很强或木神很旺，强水能崩弱土，旺木能克衰土，则干扰子丑之合化
  • 情形三（合化失败）：非但令其无法化土，反竟导致丑土崩解与子水混为流沙，增加八字格局污浊之气

吉凶判断：

  • 如果八字以水为忌，取土来制水为用，见到丑土来合子水，牵制住水势，理应论吉
  • 如果子丑再能合化成土，削弱忌神水的力量，增强用神土的力量，更是吉上加吉
  • 但若八字原局水、木两旺，破坏了子丑的合化，则虽遇丑土用神，却一点助力也没有

【实例二】寅申相冲的四柱分析

命例结构：

年柱：甲寅　　月柱：丙寅

日柱：戊申　　时柱：庚申

综合分析：

  • 干支配置：年月两寅，日时两申，形成寅申双冲之局
  • 五行分布：木火在年月（前半生），金土在日时（后半生）
  • 格局特点：寅申相冲为驿马之冲，主动荡、变迁、奔波

四柱关系判断：

  • 年月相连：甲丙相生，寅寅比助，早年木火旺相，得祖荫父母之助
  • 日时相连：戊庚相生，申申比助，晚年金土得力，子女贤孝
  • 年日相冲：甲戊相克，寅申相冲，与父母缘薄，早年离祖
  • 月时相冲：丙庚相克，寅申相冲，事业多变，中年转折

人生轨迹：此命前后半生际遇不同，早年得木火之助，中年遇金木交战，晚年归于金土安稳

【实例三】三合会局的综合判断

命例结构：

年柱：壬申　　月柱：甲辰

日柱：丙子　　时柱：戊子

地支分析：申子辰三合水局

综合判断：

  • 三合成立条件：年支申、月支辰、日支子，三字齐全，形成完整水局
  • 天干配合：年干壬水、日干丙火、月干甲木、时干戊土
  • 五行力量：地支三合水局，水势强旺；天干丙火被壬水克制，力量微弱
  • 格局判定：从势格或从旺格，需顺水之势而行

四柱相互作用：

  • 申子半合增强水势，子辰半合亦增强水势，三者形成连环相生
  • 日干丙火坐子水，失令无根，难以抵挡水势
  • 月干甲木可以通关，泄水生火，但力量有限
  • 时干戊土克水，但孤立无援，反被水势冲散

命运特征：此命水势强旺，需顺势而为，适合从事流动性行业，如贸易、运输、旅游等

【实例四】年月日时干支的完整分析

命例结构：

年柱：己卯　　月柱：戊辰

日柱：甲子　　时柱：壬申

逐柱分析：

年柱己卯分析

  • 天干己土：代表祖业根基，偏财之象
  • 地支卯木：木旺之地，克制己土
  • 干支关系：己卯干支相克，祖业难守

月柱戊辰分析

  • 天干戊土：月令正财，父母宫位
  • 地支辰土：湿土库藏，有木余气
  • 干支关系：戊辰同气，但辰中暗藏乙木克土

日柱甲子分析

  • 天干甲木：日主本人，身强力壮
  • 地支子水：正印之位，生扶日主
  • 干支关系：甲子相生，根基稳固

时柱壬申分析

  • 天干壬水：偏印食神，晚年依靠
  • 地支申金：七杀之地，子女刚强
  • 干支关系：壬申相生，金水相涵

四柱综合关系：

  • 年月关系：己戊同气，卯辰相邻不刑，土木相战但势均力敌
  • 月日关系：戊克甲为财星克身，但辰生子水，子水生甲木，形成通关
  • 日时关系：甲壬相生，子申半合水局，晚年水木相生
  • 年时关系：己壬相克，卯申暗合，首尾有情

格局判定：

  • 日主甲木生于辰月，处于余气之时
  • 年月财星透出，日时印星得力
  • 身财两停，可取财格或用印护身

运势推断：

  • 早年受财星克制，创业艰辛
  • 中年印星得用，事业转机
  • 晚年水木相生，子女孝顺，安享晚年

【实例五】刑冲会合并见的复杂命局

命例结构：

年柱：丁巳　　月柱：庚戌

日柱：己巳　　时柱：甲戌

复杂关系分析：

地支关系

  • 巳巳伏吟：年日两巳，伏吟重复，主生活刻板或有反复
  • 戌戌伏吟：月时两戌，伏吟重复，事业多有重复波折
  • 巳戌暗合：年支巳与月支戌暗合，但合而不化
  • 四库欠全：有两戌（火库），缺辰（水库）、丑（金库）、未（木库）

天干关系

  • 丁庚相合：年月天干丁壬不合，但有相克
  • 甲己相合：时干甲木合日干己土，正财合身
  • 丁己同气：年干丁火生日干己土，祖业有助

四柱综合互动：

  • 年柱与日柱：天干丁己相生，地支巳巳伏吟，与父亲缘深但关系复杂
  • 月柱与时柱：天干庚甲相冲，地支戌戌伏吟，事业多变动
  • 年柱与时柱：丁甲木火通明，巳戌暗合，首尾呼应
  • 月柱与日柱：庚克己土，戌巳不刑不冲，月令对日主有克制

综合命理判断：

  • 伏吟重见：巳巳、戌戌伏吟，主生活单调或反复经历相似之事
  • 财星合身：甲己合化土（需视全局），事业财运与配偶关系密切
  • 火土燥烈：巳火戌土多见，性格刚烈，需注意心血管系统
  • 层次分析：年月为根，日时为苗，火土相生，但过于燥热，需水调候

实际应用建议：

  • 大运流年见水，可以调候命局燥热，主转机发达
  • 大运流年见木，加重火势，主劳累辛苦
  • 适合从事火土相关行业：建筑、陶瓷、化工等
  • 需注意心脏、血压、眼睛方面的保健

五、综合分析要点总结

分析步骤 分析内容 判断要点
第一步：排盘 准确排出年月日时四柱 注意节气交界、早子夜子、真太阳时等问题
第二步：定格 判定八字格局高低 以月令为主，结合日主强弱，确定用神忌神
第三步：查冲合 查看天干地支冲合会刑穿 分析吉凶性质，判断人事应验
第四步：论强弱 判断日主与各神强弱 得令、得地、得势、得助四个方面综合判断
第五步：看配置 分析四柱配置是否得当 年月为根基，日时为发展，相互呼应为佳
第六步：察神煞 参考吉神凶煞的影响 神煞为辅助判断，不可过分依赖
第七步：推大运 结合大运流年综合判断 命局为本，运岁为变，动静结合
第八步：断吉凶 综合推断人生运势吉凶 富贵贫贱、吉凶祸福、寿夭荣枯

合盘分析的核心原则

  1. 整体观：四柱八字是一个有机整体，不可孤立看待某一柱或某一字
  2. 主次观：月令为重，日主为本，年时为辅，层次分明
  3. 动静观：命局为静，大运流年为动，动静结合方能准确
  4. 吉凶观：喜用为吉，忌仇为凶，但需结合具体情况灵活判断
  5. 变通观：命理无定法，需根据实际情况灵活变通，不可拘泥
