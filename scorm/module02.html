第二章：干支季节方位属性

天干地支的季节、方位及五行属性

学习目标

  • 掌握十天干的方位对应关系及五行属性（东南西北中）
  • 理解十二地支代表的四季时令及其季节特性
  • 学会地支的方位分布及五行阴阳属性
  • 了解四季中分法（春分、夏至、秋分、冬至）的定位原理
  • 综合运用干支的方位、季节、五行属性进行八字分析

一、天干代表的方位及属性

十天干按照五行理论，分别对应五方（东南西北中），每个方位有阴阳两个天干：

1. 甲乙木——东方

  • 甲木（阳木）：位于东方，五行属木，阳性。甲木色主青靛，譬如粗壮的乔木、大树、栋梁，象征向上生长、挺拔茂盛之力量。
  • 乙木（阴木）：位于东方，五行属木，阴性。乙木色主碧绿，譬如矮嫩的花草、瓜萝、禾麦，象征柔韧弯曲、温和生长之特性。

2. 丙丁火——南方

  • 丙火（阳火）：位于南方，五行属火，阳性。丙火色主赤朱，譬如显明的艳阳、烈火、狂风，象征炽热光明、普照万物之能量。
  • 丁火（阴火）：位于南方，五行属火，阴性。丁火色主红紫，譬如幽微的灯光、射线、核能，象征文明智慧、内敛温暖之光芒。

3. 戊己土——中央

  • 戊土（阳土）：位于中央，五行属土，阳性。戊土色主棕橙，譬如粗硬的燥土、砖瓦、沙石，象征厚重坚实、承载万物之基础。
  • 己土（阴土）：位于中央，五行属土，阴性。己土色主褐黄，譬如松软的泥田、山泽、洼地，象征柔软湿润、孕育生命之田园。

4. 庚辛金——西方

  • 庚金（阳金）：位于西方，五行属金，阳性。庚金色主灰白，譬如钢锐的锁物、钢铁、刀剑，象征刚强坚硬、肃杀收敛之气势。
  • 辛金（阴金）：位于西方，五行属金，阴性。辛金色主雪白，譬如柔脆的矿物、金银、珠玉，象征精致纯净、贵重细腻之品质。

5. 壬癸水——北方

  • 壬水（阳水）：位于北方，五行属水，阳性。壬水色深为清白、色浅为灰黑，譬如激荡的液体、江海、大雨，象征流动奔腾、汹涌澎湃之水势。
  • 癸水（阴水）：位于北方，五行属水，阴性。癸水色深为灰黑、色浅为清白，譬如平和的液体、泉露、溪流，象征滋润灌溉、细水长流之特性。

二、地支代表的季节性

十二地支按照四季时令分布，每个季节包含三个月份，对应农历的月份排列：

1. 春季（寅卯辰）

  • 寅月：初春，农历正月，万物萌动，阳气初生
  • 卯月：仲春，农历二月，草木繁茂，生机勃发
  • 辰月：季春，农历三月，春尽夏至，土旺木气余存

2. 夏季（巳午未）

  • 巳月：初夏，农历四月，阳气渐盛，暑热初现
  • 午月：仲夏，农历五月，炎炎夏日，火气最旺
  • 未月：季夏，农历六月，夏末秋初，土旺火气余存

3. 秋季（申酉戌）

  • 申月：初秋，农历七月，暑退凉生，金气初萌
  • 酉月：仲秋，农历八月，秋高气爽，金气最旺
  • 戌月：季秋，农历九月，秋尽冬临，土旺金气余存

4. 冬季（亥子丑）

  • 亥月：初冬，农历十月，寒气渐生，阴气渐长
  • 子月：仲冬，农历十一月，严寒酷冷，水气最旺
  • 丑月：季冬，农历十二月，冬尽春来，土旺水气余存

重要提示：辰、戌、丑、未四个地支本属中央土，但分派于四季之末（季春、季夏、季秋、季冬），作为四季更替转换的过渡期，各主十八天。

三、地支代表的方位及属性

十二地支除了代表季节，也对应五方方位及五行属性，且各有阴阳之分：

东方木（寅卯辰）

寅卯为东方木，寅属阳木，卯属阴木，而辰含有东方木的余气。寅卯主春天生发之气，辰为春季之库，收藏木气待来年再发。

南方火（巳午未）

巳午为南方火，巳属阳火，午属阴火（午为极阳反阴），而未含有南方火的余气。巳午主夏天炎热之气，未为夏季之库，收藏火气。

西方金（申酉戌）

申酉为西方金，申属阳金，酉属阴金，而戌含有西方金的余气。申酉主秋天肃杀之气，戌为秋季之库，收藏金气。

北方水（亥子丑）

亥子为北方水，亥属阳水，子属阴水（子为极阴反阳），而丑含有北方水的余气。亥子主冬天寒冷之气，丑为冬季之库，收藏水气。

中央土（辰戌丑未）

辰戌丑未为四季土，辰戌属阳土，丑未属阴土。四季土分别位于四季之末，起着承前启后、转换季节的作用。辰为春土（木库），未为夏土（火库），戌为秋土（金库），丑为冬土（水库）。

四、四季分法（春分、夏至、秋分、冬至）

根据太阳直射的位置，我们可以确定四季的中心点，这就是简易的四季中分法：

1. 冬至点（正子）

太阳直射南回归线时，为冬季的最中心点，我们称为"冬至点"，对应地支为正子（子月中点）。此时北半球白昼最短，黑夜最长，阴气达到极盛。

2. 夏至点（正午）

太阳直射北回归线时，为夏季的最中心点，我们称为"夏至点"，对应地支为正午（午月中点）。此时北半球白昼最长，黑夜最短，阳气达到极盛。

3. 春分点（正卯）

太阳由南往北直射赤道时，为春季的最中心点，我们称为"春分点"，对应地支为正卯（卯月中点）。此时昼夜平分，阴阳平衡，万物生发。

4. 秋分点（正酉）

太阳由北往南直射赤道时，为秋季的最中心点，我们称为"秋分点"，对应地支为正酉（酉月中点）。此时昼夜平分，阴阳平衡，万物收敛。

四季中分法的意义：通过春分、夏至、秋分、冬至四个节点，我们可以清楚地把握一年四季的阴阳消长规律。从冬至到夏至，阳气逐渐增长，阴气逐渐衰退；从夏至到冬至，阴气逐渐增长，阳气逐渐衰退。这种阴阳消长的规律，是推算命运的重要依据。

五、干支方位季节及属性综理

综合前面所学，我们可以将天干地支的方位、季节、五行属性整理如下：

1. 东方木系统

天干：甲（阳）、乙（阴）
地支：寅（阳）、卯（阴）、辰（含木余气）
季节：春季（寅卯辰合称为春季）
特性：生发向上，主生长繁茂

2. 南方火系统

天干：丙（阳）、丁（阴）
地支：巳（阳）、午（阴）、未（含火余气）
季节：夏季（巳午未合称为夏季）
特性：炎热光明，主旺盛发展

3. 中央土系统

天干：戊（阳）、己（阴）
地支：辰（阳）、戌（阳）、丑（阴）、未（阴）
季节：四季之末（辰戌丑未为四季土）
特性：承载万物，主转换过渡
特别说明：辰为春土（木库），未为夏土（火库），戌为秋土（金库），丑为冬土（水库）

4. 西方金系统

天干：庚（阳）、辛（阴）
地支：申（阳）、酉（阴）、戌（含金余气）
季节：秋季（申酉戌合称为秋季）
特性：肃杀收敛，主成熟凋落

5. 北方水系统

天干：壬（阳）、癸（阴）
地支：亥（阳）、子（阳）、丑（含水余气）
季节：冬季（亥子丑合称为冬季）
特性：寒冷藏伏，主潜藏休眠

六、实例分析

例一：判断天干五行属性

问题：在八字"甲子 丙寅 庚午 戊申"中，如何判断各天干的五行属性和方位？

分析：

  • 甲：属木，位于东方，为阳木，性质如大树栋梁，主向上生长
  • 丙：属火，位于南方，为阳火，性质如艳阳烈火，主光明炎热
  • 庚：属金，位于西方，为阳金，性质如刚铁利刃，主肃杀坚硬
  • 戊：属土，位于中央，为阳土，性质如燥土山石，主厚重承载

结论：此八字天干包含木、火、土、金四种五行，缺水。从方位看，涵盖东、南、中、西四方，缺北方。这种结构显示命主可能在北方或水相关的事业上较为欠缺。

例二：判断地支季节

问题：八字"己卯 戊辰 甲子 壬申"中的地支分别代表什么季节？

分析：

  • 卯：仲春，农历二月，春天正盛，木气最旺
  • 辰：季春，农历三月，春末夏初，土旺含木余气
  • 子：仲冬，农历十一月，寒冬正盛，水气最旺
  • 申：初秋，农历七月，秋天初至，金气初现

结论：此八字地支包含春、夏、秋、冬四季，显示命主一生经历丰富，四季变化明显。月柱和年柱都在春季，说明出生在春天，木气较旺，适合从事与木相关的行业（如文化、教育、林业等）。

例三：方位对应关系分析

问题：一个人八字为"甲寅 乙卯 丙午 丁巳"，从方位角度如何理解？

分析：

  • 年柱甲寅：甲木东方+寅木东方 = 双重东方木，且都属阳
  • 月柱乙卯：乙木东方+卯木东方 = 双重东方木，且都属阴
  • 日柱丙午：丙火南方+午火南方 = 双重南方火，阴阳组合
  • 时柱丁巳：丁火南方+巳火南方 = 双重南方火，阴阳组合

结论：此八字只有东方木和南方火，完全缺乏西方金、北方水、中央土。从方位看，命主适合向东方和南方发展，不宜往西方和北方。从五行看，木火两旺，属于"木火通明"格局，适合从事文化、艺术、科技等需要创造力和热情的行业。但需注意缺金缺水，可能在财运（金）和智慧（水）方面需要外力补充。

例四：四季土的特殊作用

问题：八字"庚辰 壬午 己未 丙戌"中有三个四季土（辰、未、戌），这有什么特殊含义？

分析：

  • 辰土：春土，木库，含有东方木的余气，位于春末
  • 未土：夏土，火库，含有南方火的余气，位于夏末
  • 戌土：秋土，金库，含有西方金的余气，位于秋末

结论：此八字三个四季土齐全（只缺冬土丑），说明命主具有很强的包容性和转换能力。辰含木气，未含火气，戌含金气，等于暗藏了木火金三行。日主己土坐未土，时柱又见戌土，土气很重，说明命主稳重踏实，但也可能过于固执。四季土多的人通常善于协调转换，能够适应不同环境，是很好的管理者或中介人才。

例五：实际八字中的方位分析应用

问题：某人八字"壬子 癸丑 甲寅 乙卯"，从季节方位角度分析其命理特点。

分析：

  • 年柱壬子：壬水北方+子水北方（仲冬），双重北方水，极寒之象
  • 月柱癸丑：癸水北方+丑土北方（季冬，含水余气），北方水土相生
  • 日柱甲寅：甲木东方+寅木东方（初春），双重东方木，生发之气
  • 时柱乙卯：乙木东方+卯木东方（仲春），双重东方木，繁茂之象

季节特点：年月柱在冬季（子丑），日时柱在春季（寅卯），从冬到春的过渡，象征由潜藏到生发。

方位特点：北方水（壬癸子丑）生东方木（甲乙寅卯），水生木的连环相生格局。

结论：此为"水木相生"的上佳格局。命主出生在冬末春初，水气充足滋养木气，如同春雨润泽大地，草木得以繁茂生长。这种人通常智慧聪明（水主智），仁慈温和（木主仁），适合从事文化教育、医疗卫生、环保农林等行业。从方位看，适合向东方和北方发展。但要注意此命缺火缺金，可能在热情（火）和决断力（金）方面稍显不足，需要后天培养。

例六：阴阳干支的配合

问题：为什么"甲子"是六十甲子的开头，而不是"甲丑"？

分析：

干支组合遵循"阳干配阳支，阴干配阴支"的原则：

  • 甲属阳木，子属阳水（虽为偶数但属性为阳），阳配阳，可以组合
  • 甲属阳木，丑属阴土，阳配阴，不能组合
  • 乙属阴木，丑属阴土，阴配阴，可以组合
  • 乙属阴木，子属阳水，阴配阳，不能组合

结论：六十甲子的排列顺序是：甲子、乙丑、丙寅、丁卯……这样阳干配阳支，阴干配阴支，每个天干都有六种组合（六甲、六乙、六丙等），共计六十组。这种阴阳配合的原理，体现了古人对自然规律的深刻认识，阴阳各有其性，相配才能和谐。

  1. 天干方位五行：
    • 甲乙木在东方（甲为阳木，乙为阴木）
    • 丙丁火在南方（丙为阳火，丁为阴火）
    • 戊己土在中央（戊为阳土，己为阴土）
    • 庚辛金在西方（庚为阳金，辛为阴金）
    • 壬癸水在北方（壬为阳水，癸为阴水）
  2. 地支季节分布：
    • 寅卯辰为春季（正二三月）
    • 巳午未为夏季（四五六月）
    • 申酉戌为秋季（七八九月）
    • 亥子丑为冬季（十十一十二月）
  3. 地支方位五行：
    • 寅卯为东方木，辰为春土含木余气
    • 巳午为南方火，未为夏土含火余气
    • 申酉为西方金，戌为秋土含金余气
    • 亥子为北方水，丑为冬土含水余气
    • 辰戌丑未为四季土，分别为木火金水之库
  4. 四季中分法：春分（正卯）、夏至（正午）、秋分（正酉）、冬至（正子）四个节点标志着四季的中心点和阴阳消长的关键时刻
  5. 干支综合运用：在实际八字分析中，要综合考虑天干地支的方位、季节、五行、阴阳等多重属性，才能准确判断命局特点
