第三章：阴阳五行通识

阴阳五行之概说

学习目标

  • 理解阴阳的哲学概念及其在命理学中的应用
  • 掌握五行（金、木、水、火、土）的基本属性和特征
  • 熟练运用五行相生相克的规律进行分析
  • 能够识别阴阳共存形态及其在实际命理中的表现
  • 综合应用阴阳五行理论进行命理推演

一、阴阳之通识

（一）阴阳的哲学概念

天干地支是四柱八字中最基本的组合符号，但若要以此干支符号来推论人命，却非得运用「阴阳五行」的理论不可。简言之，四柱八字是人命的组合方程式，而阴阳五行则是八字方程式的推演方法，至于命运吉凶现象无疑就是八字方程式的推演过程。

易经说：「干，元、亨、利、贞」。宇宙是由时、空、物、能四大要素所构成，其中时空为「阴虚」，物能为「阳实」，阴阳两者关系密切，须臾莫可相离。

阴阳的本质

宇宙天地间的万事万物，无论从任何角度的时间和空间来看，均会有某种程度的相对性。例如：

  • 昼与夜、男与女、水与火
  • 进与退、动与静、善与恶
  • 是与非、利与害、得与失
  • 生与死、吉与凶、刚与柔
  • 仁慈与暴虐、前后与左右
  • 东西与南北、富贵与贫贱
  • 快乐与悲哀

这种相对之现象，就是命理学上所谓的「阴阳」。

易经又说：「一阴一阳之谓道……阴阳不测之谓神」，又说：「昔者圣人之作易也，将以顺性命之理，是以立天之道曰阴与阳，立地之道曰柔与刚，立人之道曰仁与义」。因此，我们可以说阴阳就是一切事物相对之现象。

（二）阴阳的共存形态

易学太极论说：「太极负阴而抱阳」，经过几千年后的今天，科学终于证明任何物质中的原子结构，确属负阴抱阳（原子核内「抱」有正电荷——阳电，核外绕有「负」电荷——阴电）。由此事实我们得知如下现象：

  1. 太极负阴而抱阳，阳中有阴，阴中有阳。
  2. 阴阳交互推进引动，无法单独存在或截然分开。
  3. 太极为事物之本体，好比人命之我身；阴阳为事物之变化，好比人命之穷通。

阴阳共存的关键规律：

  • 宇宙中若无空间，则宇宙本身根本就不存在，相对的物体、能量、时间也均无法存在。
  • 宇宙空间中若没有了物体，则亦根本无时间与能量之存在。
  • 宇宙空间中若没有任何能量，则亦根本无物体与时间之存在。
  • 宇宙中若没有了时间与能量，则一切空间皆将静止，一切物体皆将死绝，宇宙势必瞬间毁灭！

由此可知，宇宙万物都有相互对立却又相互依存的本质，虽然有时会偏阴或偏阳，但绝不能孤阴孤阳单独存在。

二、五行之通识

（一）五行的哲学概念

中国的易学认为：宇宙中所存在的万事万物，不管其体积之大小或范围之广隘，均可将其分离或整合成各种单一（但不一定独立）之太极，由太极可化生两仪（即阴和阳），由两仪又可化生四象（即太阳、少阳、太阴、少阴）。

古代先贤基于此种领悟，发明出五行的哲学理论，将阴阳太极比喻成地球本体（土），再将阴阳太极所化生的四象比喻成地球内外的：

  • 木 — 植物
  • 火 — 热能
  • 土 — 地球本体（太极）
  • 金 — 矿物
  • 水 — 液体

五行的本质

中国哲学所谓的五行乃指木、火、土、金、水而言。然而，五行并非仅只代表其字面所显示之可见物象而已，要知它乃先贤为了哲学上便于研究比较起见，将宇宙万事万物之性能简单归纳成五大类，并以地球上五种有形可见之相似物为影射对象所设定之代名词。

（二）五行的属性分类

1. 五行与脏腑器官

以人的「脏腑器官」为一大太极：

  • 土 — 脾胃（中央太极）
  • 木 — 肝胆
  • 火 — 心脏
  • 金 — 肺脏
  • 水 — 膀胱肾脏

2. 五行与身体结构

以人的「身体结构」为一大太极：

  • 土 — 肌肉皮肤
  • 木 — 毛发指甲
  • 火 — 血液体温
  • 金 — 骨骼牙齿
  • 水 — 生殖内分泌机能

3. 五行与空间方位

以「空间」为一大太极：

  • 土 — 原地中央
  • 木 — 东方
  • 火 — 南方
  • 金 — 西方
  • 水 — 北方

4. 五行与人的常性美德

以人的「常性美德」（五常或五德）为一大太极：

  • 土 — 信
  • 木 — 仁
  • 火 — 礼
  • 金 — 义
  • 水 — 智

（三）五行的生克形态

「行」乃运转、流行、发展的意思，而「五行」就是指五种可运转、流行、发展的物体或质能。五行各有其阴阳正反之特性，也互有其生克制化之现象。

五行相生

金生水，水生木，木生火，火生土，土生金，循环相生。

  • 金生水：金属熔化生液体，矿物质滋养水源
  • 水生木：水份液体能帮助植物的生长
  • 木生火：植物能帮助火势的旺烈
  • 火生土：火焚烧后化为灰烬归土
  • 土生金：土壤矿藏孕育金属矿物

五行相克

金克木，木克土，土克水，水克火，火克金，间隔相克。

  • 金克木：金属利器能克伐植物
  • 木克土：树木根系破坏土壤结构
  • 土克水：土壤能吸收阻挡水流
  • 水克火：水份液体能抑制火势
  • 火克金：烈火能熔化金属

（四）五行相生相克图的文字描述

五行相生图说明：

五行相生呈顺时针循环关系：金→水→木→火→土→金，形成一个闭合的生成循环。每一行都能生出下一行，同时也被上一行所生。这种生成关系代表着相互促进、相互滋养的良性循环。

五行相克图说明：

五行相克呈五角星形关系：金克木、木克土、土克水、水克火、火克金。每一行都克制间隔的另一行，同时也被另一间隔行所克。这种克制关系代表着相互制约、相互平衡的调节机制。

注意：生与克并非绝对的好与坏，而是宇宙万物保持平衡的两种基本力量。适度的生能促进发展，适度的克能防止失衡。

四、阴阳五行实例分析

例一：阴阳判断实例

情境：判断男女与昼夜的阴阳属性

分析：

  • 男人属阳，女人属阴，有男人就必会有女人相对应
  • 男人体虽属阳，然其内却含有轻微女性荷尔蒙（阳中有阴且阳盛阴衰）
  • 女人体虽属阴，然其内却含有轻微男性荷尔蒙（阴中有阳且阴盛阳衰）
  • 因此才造成再刚强的男人也有懦弱或流泪的时候，再温柔的女人也有刚强或激愤的时候

结论：这充分体现了「太极负阴而抱阳，阳中有阴，阴中有阳」的阴阳共存原理。

例二：天干地支的阴阳属性

情境：判断甲、乙、寅、卯的阴阳属性

分析：

  • 就天干而言，甲乙属木；就地支而言，寅卯也属木
  • 在阴阳的区分上，甲、寅属阳，乙、卯则属阴
  • 四者虽同属木，然其阴阳本质却有不同
  • 甲为阳木（乔木、栋梁），乙为阴木（花草、藤蔓）
  • 寅为阳木（初春之木），卯为阴木（仲春之木）

结论：相同五行也有阴阳之分，其性质、力量、表现形式各不相同。

例三：五行相生关系分析

情境：水生木的实际应用

分析：

  • 命局中日主为甲木（属木），如果八字中水旺
  • 水能生木，则水为印绶，对日主有生扶之功
  • 水旺则木强，如同植物得到充足水分则茁壮成长
  • 但若水过旺而木弱，则为"水多木漂"，反而不利

结论：五行相生要讲究适度，过犹不及。生扶要根据命局整体平衡来判断。

例四：五行相克关系分析

情境：金克木在命理中的表现

分析：

  • 日主为乙木（阴木），八字中庚金旺盛
  • 金克木，庚金为七杀，对乙木日主形成克制
  • 如同金属利器砍伐树木，具有破坏性
  • 但若木强金弱，反而"木坚金缺"，金无法克木
  • 若有水通关（金生水，水生木），则化解克制之力

结论：五行相克要看双方力量对比，并考虑是否有通关之神化解。

例五：实际命理中的五行应用

情境：综合分析五行平衡

案例：某人生于春季（木旺之时），日主为丙火

分析：

  • 春季木旺，木能生火，对丙火日主有利
  • 但木过旺则火炎，需要金来克木，使之平衡
  • 同时需要水来调候（春季气候转暖，需水润泽）
  • 土能泄火之气，也能作为调节五行的中介
  • 整体要求金、木、水、火、土五行流通有情，不可偏枯

结论：实际命理推演需要综合考虑五行生克、季节旺衰、调候用神等多方面因素，追求五行的动态平衡。

例六：五行生克的医学应用

情境：五行与人体健康

分析：

  • 肝属木，心属火，木生火，故肝气能助心火
  • 若肝木过旺，则心火过盛，可能导致心烦失眠
  • 肺属金，肝属木，金克木，故肺气能制肝木
  • 若肺金虚弱，则无法制约肝木，导致肝阳上亢
  • 中医治疗讲究"虚则补其母，实则泻其子"

结论：五行生克理论不仅用于命理，也广泛应用于中医养生和疾病诊治。

本章总结

核心要点：

  1. 阴阳本质：阴阳是一切事物相对之现象，阴阳交互推进引动，无法单独存在或截然分开。太极负阴而抱阳，阳中有阴，阴中有阳。
  2. 五行归类：五行（木、火、土、金、水）是将宇宙万事万物之性能简单归纳成五大类的哲学理论，并非仅指字面所示之可见物象。
  3. 五行相生：金生水，水生木，木生火，火生土，土生金，循环相生。相生代表相互促进、相互滋养的良性循环。
  4. 五行相克：金克木，木克土，土克水，水克火，火克金，间隔相克。相克代表相互制约、相互平衡的调节机制。
  5. 生克应用：在实际命理推演中，要综合考虑五行生克、力量对比、通关化解、季节旺衰等因素，追求五行的动态平衡。

学习重点：

  • 深刻理解阴阳相对共存、相互依存的哲学本质
  • 熟练掌握五行相生相克的基本规律
  • 能够将五行理论应用于实际命理分析
  • 理解"过犹不及"和"动态平衡"的重要性
  • 认识到阴阳五行不仅是命理工具，更是宇宙万物运行的基本规律

延伸思考：

阴阳五行理论是中国传统文化的精髓，不仅应用于命理学，还广泛应用于中医、武术、建筑、艺术等各个领域。通过本章学习，我们应认识到这一理论的普遍性和实用性，并在实践中不断加深理解。
