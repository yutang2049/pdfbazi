第一章：干支基础知识

天干地支的基本认识

学习目标

  • 理解天干地支的起源和应用
  • 掌握六十甲子的排列规律
  • 学会四柱八字的基本排列方法
  • 认识天干地支的方位、季节及属性对应关系
  • 能够独立排列简单的四柱八字

一、天干地支的应用由来

天干地支（一称天干地枝）相传乃四千六百余年前黄帝时代的大挠氏所创（一说伏羲氏所创，世本佚文则谓：『黄帝令大挠作甲子』）。

当初创设甲、乙、丙、丁、戊、己、庚、辛、壬、癸十个天干用以记年与日，另再创设子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥十二个地支用以记月与时。

因为用十个天干记日，每月有可能出现三个同一天干之日，容易造成混淆，因此古代先知遂又将十天干与十二地支按「奇干配奇支，偶干配偶支」的方式组合起来合称「干支」，计得六十组。

备注：

  • 一旬有十组干支，而每旬所辖之十组干支组合，均自甲干起至癸干止，故六十甲子共计有六旬。
  • 天干地支之组合乃取奇数配奇数，偶数配偶数；故有甲子却无甲丑，有乙丑却无乙子，有丙寅却无丙卯，有丁卯却无丁寅……。

三、四柱八字的排列方法

在我们利用干支推算命运的时候，其记年、记月、记日、记时的干支各为一柱，合计为四柱，每柱天干地支各一字，四柱干支共计八个字，故一般人都习惯称之为「排四柱」或「算八字」。

实例一：传闻黄帝开国

传闻黄帝开国于今四千六百八十二年前的甲子年、甲子月、甲子日、甲子时，其四柱八字为：

年柱 月柱 日柱 时柱
甲子 甲子 甲子 甲子

横列式：甲子、甲子、甲子、甲子

实例二：蒋公中正

传闻蒋公中正出生于民国廿五年（光绪十三年）农历九月十五日午时（一说未时），则依午时为准排列，其四柱八字为：

年柱 月柱 日柱 时柱
丁亥 庚戌 己巳 庚午

横列式：丁亥、庚戌、己巳、庚午

实例三：于右任先生

党国元勋于右任先生据闻出生于民国卅三年（光绪五年）农历二月二十日申时，其四柱八字为：

年柱 月柱 日柱 时柱
己卯 戊辰 甲子 壬申

横列式：己卯、戊辰、甲子、壬申

详细实例分析

例题一：六十甲子的组合规律

问题：为什么六十甲子中有甲子而没有甲丑？

分析步骤：

  1. 天干分为阳干（甲、丙、戊、庚、壬）和阴干（乙、丁、己、辛、癸）
  2. 地支分为阳支（子、寅、辰、午、申、戌）和阴支（丑、卯、巳、未、酉、亥）
  3. 干支组合遵循"奇干配奇支，偶干配偶支"的原则
  4. 甲为阳干（奇数），子为阳支（奇数），可以配对
  5. 甲为阳干（奇数），丑为阴支（偶数），不能配对

结论：因为甲属阳干，只能与阳支配对，子为阳支可配，丑为阴支不可配，所以有甲子而无甲丑。

例题二：判断干支的阴阳属性

问题：如何快速判断"庚申"这个干支组合是否正确？

分析步骤：

  1. 查找庚在十天干中的位置：甲、乙、丙、丁、戊、己、庚（第七位）
  2. 第七位为奇数位，庚属阳干
  3. 查找申在十二地支中的位置：子、丑、寅、卯、辰、巳、午、未、申（第九位）
  4. 第九位为奇数位，申属阳支
  5. 阳干配阳支，符合组合规律

结论："庚申"是正确的干支组合，在六十甲子中排第57位。

例题三：四柱排列的基本原则

问题：某人出生于甲子年、乙丑月、丙寅日、丁卯时，如何正确排列四柱？

分析步骤：

  1. 四柱排列顺序为：年柱、月柱、日柱、时柱
  2. 年柱：记录出生年份的干支 → 甲子
  3. 月柱：记录出生月份的干支 → 乙丑
  4. 日柱：记录出生日期的干支 → 丙寅
  5. 时柱：记录出生时辰的干支 → 丁卯

结论：四柱八字为"甲子 乙丑 丙寅 丁卯"，横列式书写，从左至右依次为年月日时。

例题四：六十甲子的周期循环

问题：如果今年是甲子年，那么六十年后是什么年？一百二十年后又是什么年？

分析步骤：

  1. 六十甲子是一个完整的循环周期
  2. 从甲子开始，到癸亥结束，共计60组
  3. 60年后，干支重新回到起点
  4. 60年后：甲子年 + 60年 = 甲子年
  5. 120年后：甲子年 + 120年 = 甲子年 + (60×2)年 = 甲子年

结论：六十年后仍是甲子年，一百二十年后也是甲子年。这就是"花甲"、"周甲子"的含义。

例题五：天干地支数量关系

问题：十天干与十二地支组合，为什么只能产生60组而不是120组？

分析步骤：

  1. 如果不限制配对规则，10×12 = 120种组合
  2. 但因为采用"奇干配奇支，偶干配偶支"的规则
  3. 5个阳干只能配6个阳支 = 30组
  4. 5个阴干只能配6个阴支 = 30组
  5. 总计：30 + 30 = 60组
  6. 也可以从最小公倍数角度：10和12的最小公倍数是60

结论：因为干支配对遵循阴阳匹配原则，所以只能产生60组组合，而不是120组。

本章小结

本章介绍了天干地支的基本知识，这是学习四柱八字的基础。主要内容包括：

  1. 天干地支的起源：相传由黄帝时代的大挠氏所创，已有四千六百余年历史。十天干用以记年与日，十二地支用以记月与时。
  2. 六十甲子：天干地支按"奇干配奇支，偶干配偶支"的方式组合，形成六十组干支，称为"六十甲子"或"一周甲子"，周而复始，循环不绝。
  3. 四柱八字：记年、记月、记日、记时的干支各为一柱，合计四柱，每柱天干地支各一字，共八个字，这是推算命运的基本依据。
  4. 干支属性：天干地支各有其对应的方位、五行、阴阳属性和季节特征。东方木、南方火、西方金、北方水、中央土，这是五行方位的基本对应关系。
  5. 实践应用：通过黄帝、蒋公中正、于右任等实例，了解了四柱八字的实际排列方法，为后续深入学习打下基础。

掌握本章内容是学习四柱八字的第一步，务必熟记十天干、十二地支及其对应的属性关系。
